🍽️ Recipe Tracker – Full Stack Web Application

Recipe Tracker is a full-stack web application that allows users to create, manage, and organize their recipes efficiently.
This project is built with a strong backend architecture while keeping the frontend simple and clean using HTML and CSS.

The main focus of this project was backend development, API design, and database handling.

🚀 Tech Stack
Frontend

HTML5

CSS3

JavaScript

React (Vite)

Backend

Node.js

Express.js

MongoDB

REST APIs

📌 Features

Add new recipes

View all recipes

Update existing recipes

Delete recipes

Structured backend API

Database integration with MongoDB

Clean and responsive UI# Reciper_Tracking
